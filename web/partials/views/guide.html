Getting Started with Kronforce

You're up and running. Follow these steps to build your first automation.

1

Create Your First Job

Jobs are automated tasks that run on a schedule, on events, or on demand. Start with something simple:

⚡ Shell Command
echo "Hello from Kronforce"
🌐 HTTP Health Check
GET https://httpbin.org/get
2

Trigger and Watch

Click the ▶ Trigger button on any job. Then check Executions to see the result with output, exit code, duration, and timestamps.

3

Get Alerted When Things Break

Configure notifications so you know immediately when a job fails. Set up one or more channels:

Slack
Email
PagerDuty
Teams
SMS

Then enable On failure in any job's Alerts tab.

4

Organize with Groups

Group related jobs together: ETL, Monitoring, Deploys, Maintenance. Groups appear as cards on the Groups page and as filters on the Jobs list.

5

Chain Jobs into Workflows

Build powerful automation by connecting jobs together:

→ Dependencies — Job B runs only after Job A succeeds
→ Event triggers — react to completions, failures, or output patterns
→ Output extraction — capture values from stdout, pass to the next job via {{VAR}}
6

Run Jobs on Remote Machines

Distribute work across your infrastructure with agents. Start a standard agent:

KRONFORCE_CONTROLLER_URL=http://this-server:8080 \
KRONFORCE_AGENT_KEY=kf_your_agent_key \
KRONFORCE_AGENT_NAME=my-agent \
cargo run --bin kronforce-agent

Or write a custom agent in Python, Go, or Node — see examples in the repo.

7

Lock It Down

🔑 API key scoping — restrict keys to specific groups
🔒 SSO/OIDC — Okta, Azure AD, Google login
✅ Approval gates — require sign-off on critical jobs
⏰ SLA deadlines — alert when jobs run late
🔐 Secret variables — masked credentials
8

Monitor Everything

📊 Dashboard — timeline, charts, stats
📈 Prometheus — scrape /metrics
🔔 Events — activity feed + severity filters
📝 Audit log — who changed what, when
💚 Health — /api/health for load balancers
Need more detail? Check the full documentation or visit GitHub for issues and discussions.
